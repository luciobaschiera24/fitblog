# rutinas.hurlingham 💪

Blog de rutinas de gimnasio desarrollado con Django y Python.

## Descripción
Página web para compartir rutinas de entrenamiento. Los usuarios pueden registrarse, crear rutinas, editarlas, borrarlas y enviarse mensajes entre sí.

## Tecnologías utilizadas
- Python
- Django
- HTML

## Instalación

1. Clonar el repositorio
2. Crear entorno virtual: `python -m venv venv`
3. Activar entorno virtual: `venv\Scripts\activate`
4. Instalar dependencias: `pip install -r requirements.txt`
5. Aplicar migraciones: `python manage.py migrate`
6. Crear superusuario: `python manage.py createsuperuser`
7. Correr servidor: `python manage.py runserver`

## Funcionalidades
- Registro y login de usuarios
- Perfil de usuario con avatar y biografía
- CRUD completo de rutinas
- Buscador de rutinas
- Mensajería entre usuarios
- Panel de administración
